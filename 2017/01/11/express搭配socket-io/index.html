express搭配socket.io

今天给大家分享的主要是关于socket.io如何使用的一篇文章，讲述了express中socket.io如何操作

首先是引包


这相当于是一个公式

然后在客户端页面编辑


1.客户端页面中用下面的语句引入秘密文件
<script type="text/javascript" src="/socket.io/socket.io.js"></script>
2.获取socket对象
var socket = io();
3.用下面的语句发出消息，miyao是客户端自定义的通信秘钥
socket.emit('miyao',function(msg))
4.用下面的语句监听服务器的消息，liaotian是服务器自定义的秘钥，只有客户端和服务器的秘钥一致才能收到服务器的消息
socket.on("liaotian",function(msg){})

最后在服务器页面中编辑


1.监听connection事件，获取socket对象
io.on("connection",function(socket){...}
2.监听客户端发送过来的消息，“liaotian”是通信秘钥
socket.on("miyao",function(msg){}
3.服务器广播消息
io.emit('liaotian',msg)

最后

用socket.io可以做聊天室，qq等应用，
聊天室的原理就是，很多客户端连接到同一个服务器路由下，然后服务器广播每个客户端发送过来的消息并显示在每个客户端的界面上，这样每个客户端就可以看见其他客户端发送的消息了，这是多对多聊天，一对一聊天就是两个客户端进入一个服务器路由然后服务器广播彼此的信息即可
